虚拟机的搭建过程

第一次接触虚拟机这个东西，跟着网上的教程，少走了很多弯路，由于是第一次搭建不太熟悉，但还是完成了任务。

一、下载虚拟机运行软件以及Ubuntu桌面系统
我这边准备下载的是VMware Workstation Pro 17，也是目前 Windows 最主流的虚拟机软件。
分别在其官网上下载好之后，准备开始安装。

二、安装虚拟机
根据提示慢慢走就好了，许可证之类的东西可以在网上找，很多都能用。

三、创建Ubuntu 系统虚拟机
1.打开刚刚安装好的 VMware Workstation Pro 17，进入软件后，点击“创建新的虚拟机”。
2.选择“自定义（高级）”
3.选择“安装程序光盘映像文件”，点击输入框旁边的“浏览”按钮，根据下载路径，找到下载好的 Ubuntu 镜像安装包并选择。
4.如果能看到标注的小字（已检测到Ubuntu 64位 22.04.4），说明安装包是对的，并且可以正常安装，这时点击“下一步”。
5.根据网上的教程，他推荐我每个处理器的内核数量修改成2，内存选择4GB，以及网络类型选择桥接，还有硬盘容量我选择了60GB因为的我的盘符还有180G，我就弄多点了，自此我们把 Ubuntu 的虚拟机创建好了，但还没安装此系统，正式的安装需要启动虚拟机。

四、Ubuntu 系统安装过程的配置
1.虚拟机创建好后，进入虚拟机。
2.本着学习的态度，我选择了全英文的开发环境。
3.之后就是next + install now + continue一路过，时区选择shanghai
4.安装完成后 Restart Now 重启
5.初次进入 Ubuntu 系统会有一个欢迎界面，内容忽略，直接点击右上角的“Skip”跳过。
这里是提示是否要升级成 Ubuntu Pro，根据网上的教程建议默认选择“Skip for now”，然后点击“Next”。然后是 Ubuntu 系统的改善建议，改成“No, don’t send sysytem info”，然后点击“Next”。最后一步直接“Done”跳过，自此Ubuntu22.04虚拟机搭建完毕。
